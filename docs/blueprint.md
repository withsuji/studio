# **App Name**: FocusFlow

## Core Features:

- Dashboard Overview: Displays key metrics like focus time percentage, projected revenue, and pending invoices.
- Scheduling Link: Allows external users to book meetings, including timezone conversion and availability display.
- Calendar & Focus Time: Manages personal schedules and visually distinguishes focus blocks from regular meetings.
- Time Entry Management: Converts calendar events into billable time entries, with toggle for billing status.
- Invoice Generation: Bundles billable time entries into a previewable invoice.
- Billable Entry Detection: Automatically flags calendar events as billable based on keywords (tool).
- Data Persistence: Stores and retrieves user profiles, calendar events, and time entries using Firestore.

## Style Guidelines:

- Primary color: Deep indigo (#4F46E5) to convey focus and professionalism.
- Background color: Light gray (#F9FAFB) for a clean and spacious feel.
- Accent color: Teal (#14B8A6) to highlight key actions and focus blocks.
- Body font: 'Inter', a grotesque-style sans-serif font. Headline Font: 'Space Grotesk', a proportional sans-serif font for emphasis.
- Use 'Lucide React' icons for a consistent and modern look.
- Spacious layout with generous padding and margins to enhance readability.
- Subtle animations for transitions and loading states to improve UX.
